For Gnome/Ubuntu(to be able to copy/paste outside vim clipboard, you need to install ```sudo apt install vim-gtk3```)

You can also check this with ```vim --version | grep clip```

```
set number
syntax on
set ttymouse=xterm2
set mouse=a
set clipboard=unnamedplus
set ignorecase
set hlsearch
set incsearch
set noswapfile
colorscheme pablo
set t_Co=256
```
